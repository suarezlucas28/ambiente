Acceso

 

Para acceder al sistema, deberá introducir el usuario y contraseña que le fue

asignado y en cada oportunidad para avanzar a cada campo de texto, deberá presionar ENTER.

Ingresar Usuario y presionar Enter.

 

 

Ingresar Contraseña y presionar Enter.

 

 

Presionar Botón Aceptar.

 

En caso de presionar ENTER con el campo de texto vacío,

el sistema lanzará el siguiente mensaje.

 

 

En caso de que la contraseña sea errónea pero el usuario sea correcto,

el sistema lanzará el siguiente mensaje.

 

 

Luego de 3 intentos fallidos el sistema se cerrara y el usuario de bloqueara.

para desbloquar un usuario, accede a Permisos

 

Si el usuario que intenta acceder no existe,

el sistema lanzará el siguiente mensaje.

 

 

En caso de que el ingreso al sistema sea exitoso,

el sistema mostrará la pantalla de menu.

 

En caso de que el usuario vea estos mensajes, deberá contactar

con el desarrollador del sistema, Sr. Lucas Suárez.
