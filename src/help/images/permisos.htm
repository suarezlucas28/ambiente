Permisos

 

En esta pantalla podrá gestionar permisos de los usuarios, como asi también desbloquear

los que se encuenetren bloqueados.

 

GESTIONAR:

 

Presionar el botón GESTIONAR.

 

 

El sistema habilita la lista desplegable para seleccionar el usuario.

EL mismo se selecciona con el Raton, haciendo click o con las teclas de flecha arriba y flecha abajo.

Luego para avanzar se presiona ENTER.

 

 

El sistema habilita la grilla o tabla y el boton REGISTRAR.

Primeramente se deberá seleccionar los permisos a otorgar, seleccionando con el ratón cada casilla de activación.

Las casillas nos marcadas, seran tomadas como accesos denegados o prohibidos

Luego presionar el botón REGISTRAR.

 

 

DESBLOQUEAR USUARIOS:

 

Presiona el botón DESBLOQUEAR.

 

 

Seleccionar con el Ratón el usuario a desbloquear de la grilla o tabla, haciendo Click.

El sistema lanzará un mensaje.

 

 

Presionar el botón DESBLOQUEAR.

 

 

SI para desbloquear, NO para cancelar.
