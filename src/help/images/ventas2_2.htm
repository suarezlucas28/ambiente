Ventas

 

En esta pantalla podrá generar y anular ventas.

La venta dentro del sistema, representa la forma principal de disminuir en forma remunerada el STOCK de los Artículos.

 

GENERAR:

 

Presionar el botón nuevo, el sistema generará un nuevo

número de venta y habilitará los comandos de Contado o Crédito.

 

 

Selecionar uno de los comandos.

Si el comando selecionado fue el de Contado.

 

 

El sistema habilitará el campo Artículo.

 

Caso contrario, si el comando seleccionado fue el de Crédito.

 

 

El sistema habilitará la lista desplegable, para seleccionar el cliente.

 

El mismo se selecciona con el Ratón, haciendo click o con las teclas de flecha arriba y flecha abajo.

Luego para avanzar se presiona ENTER.

 

 

El sistema habilitará el campo Artículo.

 

En caso de que el el cliente no se encuentre en el sistema, podrá agregarlo,

presionando el botón que se encuentra a la derecha inmediata de la lista de clientes.

 

 

Para ingresar un nuevo cliente, aquí Clientes.

 

Luego de agregarlo deberá actualizar la lista con el botón ubicado a la derecha inmediata.

 

 

Ingresar artículos a la venta:

 

Una vez que el sistema habilita el campo Artículo, este campo no permite introducir ningún tipo de caracter.

Solo permite una accion, la de presionar la tecla ENTER.

 

Presionar ENTER.

 

 

El sistema muestra un buscador para seleccionar los productos que formaran parte

del detalle de la compra, en dicho buscador, solo se listarán los productos que han

sido cargados para dicho proveedor.

 

Puede utilizar el filtro escribiendo fragmentos de la descripción del

producto buscado.

 

 

Para Selecionar el Artículo puede hacerlo de dos maneras,

la primera es seleccionar el registro a modificar desde la grilla o tabla del buscador con el Ratón.

 

 

La segunda es hacer FLECHA ABAJO (del teclado) varias veces hasta ubicarse sobre el Artículo que desea

y presionar ENTER.

 

 

Al selecionarlo, el sistema ocultará el buscador y cargará la descripción del producto y posicionará al usuario sobre el control de FORDO/UNIDAD.

 

 

En dicho control, no hace falta borrar, solo presionar la tecla F o f, si es que la venta de ese producto se realiza por Fardo,

de la misma manera presionar la tecla U o u, si es que la venta de ese producto se realiza por Unidad.

 

En ambos casos luego de presionar dichas teclas, el usuario deberá presionar la tecla ENTER, el sistema completará el campo del precio

asignado a ese producto ya sea por fardo o unidad, y habilitará el campo de PRECIO.

 

 

Dicho campo esta destinado a modificar si se desea el precio de venta de los artículos solo acepta caracteres numéricos y el punto decimal (.).

Luego para avanzar se presiona ENTER.

 

 

 

Dicho campo solo acepta caracteres numéricos y cantidades mayores a 0.

Luego para avanzar se presiona ENTER.

 

 

El artículo es cargado a la venta exitosamente.

El sistema posicionará nuevamente el cursor el campo de artículo, permitiendo agregar otro producto

realizando el mismo procedimiento recién demostrado.

 

El sistema también habilitará los botones de GRABAR, ELIMINAR y los comandos de descuentos.

 

 

 

Si algún producto fue cargado de forma errónea, el usuario podrá eliminarlo del contenido de la compra,

seleccionandolo de la grilla con el botón izquierdo del ratón y presionando el botón Eliminar.

 

 

En caso que no haya stock suficiente, el sistema lanzará el siguiente mensaje.

 

 

Para terminar, presionar el botón Grabar.

 

En el caso de ser a crédito, el sistema consultará si la venta incluye

monto de entrega, si responde afirmativamente, pedirá que se ingrese un monto,

y se generará un cobro en forma automática.

 

 

En el caso de ser al contado, se generará un cobro en forma automática.

 

En ambos casos al finalizar veremos el siguiente mensaje.

 

 

 

Todas las operaciones podrán ser canceladas

presionando el botón CANCELAR

 

ANULAR:

 

Presionar el botón anular que se encuentra en la pantalla principal,

esto abrirá la siguiente ventana.

 

 

En la cual se encuentran listados todas las ventas registradas.

Para anular una, deberá seleccionar con el botón izquierdo del ratón

la deseada, o presionar repetidamente la tecla FECHA ABAJO del teclado

hasta posicionarse en la venta deseada y presionar ENTER, esto dará el siguiente mensaje.

 

 

Presionar aceptar, posteriormente presionar el botón

anular, cuando se haya anulado correctamente,

el sistema mostrará el siguiente mensaje.

 

 

 

En casos de que la venta tenga anidada algún cobro, este también será anulado.

 

Todas las operaciones podrán ser canceladas

presionando el botón CANCELAR
