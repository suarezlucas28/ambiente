Proveedores

 

En esta pantalla podrá agregar, modificar y borrar los Proveedores.

En todos los casos para avanzar de campo de texto, deberá presionar ENTER.

 

AGREGAR:

 

Presionar el botón Agregar.

 

 

Campo de texto, solo acepta caracteres de tipo alfabético.

Rellenar y presionar ENTER.

 

 

Campo de texto, solo acepta caracteres de tipo alfanuméricos.

Rellenar y presionar ENTER.

 

 

Campo de texto, solo acepta caracteres de tipo alfanuméricos.

Rellenar y presionar ENTER.

 

 

Campo de texto, solo acepta caracteres de tipo alfanumérico.

Rellenar y presionar ENTER.

 

 

Habilita botón Grabar.

Presionar, para terminar.

 

 

Mensaje de operación exitosa.

 

Si antes de grabar, desea modificar algun campo nuevamente, hacer click sobre el mismo, el cursor se ubicará

en dicho campo, permitiendo la edición.

 

 

Si no se llegase a conocer todos los datos del proveedor dejarlos en blanco,

presionando tecla ESPACIO y luego ENTER, se recomienda, rellenar si o si

el NOMBRE.

 

MODIFICAR:

 

Presionar botón Modificar.

 

 

Seleccionar el registro a modificar desde la grilla o tabla de la derecha con el Ratón,

o presionar repetidamente la tecla FECHA ABAJO del teclado hasta posicionarse

en el registro deseado y presionar ENTER.

.

 

El sistema cargara los campos modificables.

De esta manera, podrá modificar todos los datos que desee,

para pasar de campo de texto, deberá presionar ENTER.

 

 

Habilita botón Grabar.

Presionar, para terminar.

 

 

Mensaje de operación exitosa.

 

Si llegado al caso quisiera editar solo un campo, hacer click sobre el mismo, el cursor se ubicará

en dicho campo, permitiendo la edición.

 

 

Si no se llegase a conocer todos los datos del proveedor dejarlos en blanco,

presionando tecla ESPACIO y luego ENTER, se recomienda, rellenar si o si

el NOMBRE.

 

BORRAR:

 

Presionar botón Borrar.

 

 

Seleccionar el registro a borrar desde la grilla o tabla de la derecha con el Ratón,

o presionar repetidamente la tecla FECHA ABAJO del teclado hasta posicionarse

en el registro deseado y presionar ENTER, esto dará el siguiente mensaje.

 

 

SI para borrar, NO para cancelar.

Opción NO:

 

 

En caso de presionar ENTER con el campo de texto vacío,

el sistema lanzará el siguiente mensaje.

 

 

Todas las operaciones podrán ser canceladas

presionando el botón CANCELAR
