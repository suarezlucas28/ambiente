Ajustes

 

En esta pantalla podrá generar y anular ajustes de Stock.

Dichos ajustes sirven para aumentar o disminuir el STOCK de los artículos por fardo o unidad

sin generar compras o ventas.

 

GENERAR:

 

Presionar el botón "Nuevo".

 

 

El sistema genera un nuevo código de ajuste y habilitará los comandos para seleccionar el tipo de operación

a realizar, selecionar la operación que desee, Aumentar o Disminuir.

 

 

El sistema habilitará los comandos para seleccionar si el articulo sera tratado como fardo o unidad,

selecionar el comando que desee, Fardo o Unidad.

 

 

El sistema habilita el campo de motivo, en el cual el usuario deberá introducir el motivo por el cual realiza el ajuste.

 

Campo Alfanumérico, rellenar y presionar ENTER.

 

 

El sistema habilitará el campo de Artículo, este campo no permite introducir ningún tipo de caracter.

Solo permite una accion, la de presionar la tecla ENTER.

 

Presionar ENTER.

 

 

El sistema muestra un buscador para seleccionar los productos que formaran parte

del detalle del ajuste, en dicho buscador, solo se listarán todos productos que han

sido cargados en el sistema.

 

Puede utilizar el filtro escribiendo fragmentos de la descripción del

producto buscado.

 

 

Para Selecionar el Artículo puede hacerlo de dos maneras,

la primera es seleccionar el registro a modificar desde la grilla o tabla del buscador con el Ratón.

 

 

La segunda es hacer FLECHA ABAJO (del teclado) varias veces hasta ubicarse sobre el Artículo que desea

y presionar ENTER.

 

 

Al selecionarlo, el sistema ocultará el buscador y cargará la descripción del producto,

habilitando el campo para ingresar la cantidad de Fardos/Unidades a ajustar.

 

 

El sistema posicionará el cursor en el campo de texto Cantidad,

dicho campo solo acepta caracteres numéricos y cantidades mayores a 0.

Luego para avanzar se presiona ENTER.

 

 

El artículo es cargado a la compra exitosamente.

El sistema posicionará nuevamente el cursor el campo de artículo, permitiendo agregar otro producto

realizando el mismo procedimiento recién demostrado.

 

El sistema también habilitará los botones de GRABAR y ELIMINAR ARTÍCULO.

 

 

 

Si algún producto fue cargado de forma errónea, el usuario podrá eliminarlo del contenido del ajuste,

seleccionandolo de la grilla con el botón izquierdo del ratón y presionando el botón Eliminar Artículo.

 

 

Para terminar, presionar el botón Grabar.

 

 

Todas las operaciones podrán ser canceladas

presionando el botón CANCELAR

 

 

ANULAR:

 

Presionar el botón Anular.

 

 

En la cual se encuentran listados todos los ajustes registrados.

Para anular uno, deberá seleccionar con el botón izquierdo del ratón

el deseado, o presionar repetidamente la tecla FECHA ABAJO del teclado

hasta posicionarse en el ajuste deseado y presionar ENTER, esto dará el siguiente mensaje.

 

 

Presionar aceptar, posteriormente presionar el botón

anular, cuando se haya anulado correctamente,

el sistema mostrará el siguiente mensaje.

 

 

 

Todas las operaciones podrán ser canceladas

presionando el botón CANCELAR
