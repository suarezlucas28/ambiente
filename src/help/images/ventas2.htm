Compras

 

En esta pantalla podrá generar y anular Compras.

La compra dentro del sistema, representa la forma principal de aumentar el STOCK de los Artículos.

 

GENERAR:

 

Presionar el botón Nueva, el sistema generará un nuevo

número de compra y habilitará los comandos de CONTADO/CRÉDITO.

 

 

Estos comandos puede ser seleccionado con el Ratón o presionando Enter sobre el comando

marcado (es el que tiene recuadro) previamente.

 

Seleccionar el tipo de compra (contado o crédito).

 

 

El Sistema habilita nuevos comandos para la modalidad de compra de los artículos (Fardo/Unidad).

Estos comandos puede ser seleccionado con el Ratón o presionando Enter sobre el comando

marcado (es el que tiene recuadro) previamente.

 

 

Selecionar la modalidad de compra de los artículos.

 

 

 

El sistema habilita la lista desplegable, para seleccionar el proveedor.

 

El mismo se selecciona con el Ratón, haciendo click o con las teclas de flecha arriba y flecha abajo.

Luego para avanzar se presiona ENTER.

 

 

El sistema habilitará el campo de Artículo, este campo no permite introducir ningún tipo de caracter.

Solo permite una accion, la de presionar la tecla ENTER.

 

Presionar ENTER.

 

 

El sistema muestra un buscador para seleccionar los productos que formaran parte

del detalle de la compra, en dicho buscador, solo se listarán los productos que han

sido cargados para dicho proveedor.

 

Puede utilizar el filtro escribiendo fragmentos de la descripción del

producto buscado.

 

 

Para Selecionar el Artículo puede hacerlo de dos maneras,

la primera es seleccionar el registro a modificar desde la grilla o tabla del buscador con el Ratón.

 

 

La segunda es hacer FLECHA ABAJO (del teclado) varias veces hasta ubicarse sobre el Artículo que desea

y presionar ENTER.

 

 

Al selecionarlo, el sistema ocultará el buscador y cargará la descripción del producto y el costo del mismo,

habilitando el campo para ingresar la cantidad de Fardos/Unidades Compradas.

 

 

Dicho campo solo acepta caracteres numéricos y cantidades mayores a 0.

Luego para avanzar se presiona ENTER.

 

 

El artículo es cargado a la compra exitosamente.

El sistema posicionará nuevamente el cursor el campo de artículo, permitiendo agregar otro producto

realizando el mismo procedimiento recién demostrado.

 

El sistema también habilitará los botones de GRABAR, ELIMINAR y los comandos de descuentos.

 

 

 

 

Si algún producto fue cargado de forma errónea, el usuario podrá eliminarlo del contenido de la compra,

seleccionandolo de la grilla con el botón izquierdo del ratón y presionando el botón Eliminar.

 

 

Para terminar, presionar el botón Grabar.

 

Si la compra, fue realizada al contado, se visulizará inmediátamente el siguiente mensaje.

 

 

Selecionar la Forma de Pago.

El sistema mostrará el siguiente mensaje.

 

 

En cambio si la compra fue realizada a crédito, el sistema mostrará la siguiente ventana.

 

 

Escoger entre, "Con entrega" y "Sin entrega"

 

En el caso de Con Entrega, ingresar el monto de entrega y presionar Enter.

 

 

Se visualiza el siguiente mensaje.

 

 

Selecionar la Forma de Pago.

El sistema mostrará el siguiente mensaje.

 

 

En el caso de Sin entrega, se visulizará el siguiente mensaje.

 

 

Si el proveedor ha realizado un descuento en la compra, el sistema le permitirá registrarlo,

el usuario deberá precionar el comando SI de los descuentos.

 

 

El sistema habilitrá el campo para ingresar el descuento, dicho campo es numérico y acepta el punto decimal (.).

Ingresar el monto de descuento y presionar la tecla ENTER.

 

 

El descuento será restado del total de la venta y el sistema posicionará el curso sobre el botón grabar,

Se recomienda realizar los descuentos después de haber cargado todo el detalle de la compra.

 

Para Cancelar el Descuento, solo bastará con presionar nuevamente el comando NO.

 

En caso de tratar de ingresar artículos que ya formen parte de la compra,

el sistema lanzará un mensaje como este.

 

 

Todas las operaciones podrán ser canceladas

presionando el botón CANCELAR

 

Las compras al contado y a crédito con entregas, suponen un pago a proveedor.

 

ANULAR:

 

Presionar el botón anular que se encuentra en la pantalla principal,

esto abrirá la siguiente ventana.

 

 

 

En la cual se encuentran listadas todas las Compras.

Para anular una, deberá seleccionar con el botón izquierdo del ratón

la deseada, o presionar repetidamente la tecla FECHA ABAJO del teclado

hasta posicionarse en la compra deseada y presionar ENTER, esto dará el siguiente mensaje.

 

 

Posteriormente presionar el botón anular, cuando se haya anulado correctamente,

el sistema mostrará el siguiente mensaje.

 

 

 

En casos de que la compra tenga anidada algún pago, este también será anulado.

 

Todas las operaciones podrán ser canceladas

presionando el botón CANCELAR
