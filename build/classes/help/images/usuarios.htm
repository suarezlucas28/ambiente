Usuarios

 

En esta pantalla podrá agregar, modificar y borrar usuarios.

 

AGREGAR:

 

Presionar el botón agregar.

 

 

Campo de texto, acepta caracteres alfabéticos.

Rellenar y presionar ENTER.

 

 

Campo de texto, acepta todo tipo de caracteres.

Rellenar y presionar ENTER.

 

 

Habilita botón Guardar.

Presionar, para terminar.

 

 

MODIFICAR:

 

Presionar botón Modificar.

 

 

Seleccionar con el Ratón el registro a modificar de la grilla o tabla, haciendo Click.

El sistema rellenara los campos modificales.

 

 

De esta manera, podrá modificar todos los datos que desee,

para pasar de campo de texto, deberá presionar ENTER, el proceso es similar al de AGREGAR.

 

 

Habilita botón Grabar.

Presionar, para terminar.

 

 

Mensaje de operación exitosa.

 

BORRAR:

 

Presionar botón Borrar.

 

 

Seleccionar con el Ratón el registro a modificar de la grilla o tabla, haciendo Click.

El sistema rellenara los campos con los datos y el sistema lanzará el

siguiente cuadro de seguridad.

 

 

SI para borrar, NO para cancelar.

 

Opción SI:

 

 

Opción NO:

 

 

En caso de presionar ENTER con el campo de texto vacío,

el sistema lanzará el siguiente mensaje.

 

 

Todas las operaciones podrán ser canceladas

presionando el botón CANCELAR
